Second-Order Difference Equations

This section concerns equations of the form

ayn+2+byn+1+cyn=fn,ay_{n+2}+by_{n+1}+cy_{n} = f_n,ayn+2​+byn+1​+cyn​=fn​,

also known as recurrence relations of order 2. The method we can use for solving these is extremely similar to the one we used for constant-coefficient DEs: eigenfunctions.

Method. (Eigenfunctions for difference equations).

Suppose that some operator D[yn]=ynD[y_n] = y_nD[yn​]=yn​ relates yny_nyn​ to yn+1y_{n+1}yn+1​. As such, the above equation can be rewritten as:

aD2[yn]+bD[yn]+cyn=fnaD^2[y_n] + bD[y_n] + cy_n = f_naD2[yn​]+bD[yn​]+cyn​=fn​

The eigenfunction of this operator is yn=λny_n = \lambda^nyn​=λn for some constant λ\lambdaλ. This is because, by definition, yn+1=λn+1y_{n+1} = \lambda^{n+1}yn+1​=λn+1 (simply by substituting nnn with n+1n+1n+1), and thus

D[yn]=yn+1=λn+1=λynD[y_n] = y_{n+1} = \lambda^{n+1} = \lambda y_nD[yn​]=yn+1​=λn+1=λyn​

which is a constant multiple of yny_nyn​.

Using the exact same approach as we applied to differential equations, we solve the corresponding homogeneous difference equation aD2[yn]+bD[yn]+cyn=0aD^2[y_n] + bD[y_n] + cy_n = 0aD2[yn​]+bD[yn​]+cyn​=0 first by substituting in the eigenfunction yn=λny_n = \lambda^nyn​=λn:

aλn+2+bλn+1+cλn=0  ⟺  aλ2+bλ+c=0a\lambda^{n+2} + b\lambda^{n+1} + c\lambda^n = 0 \iff a\lambda^2 + b\lambda + c = 0aλn+2+bλn+1+cλn=0⟺aλ2+bλ+c=0

yielding a characteristic equation of degree 2. Similarly to differential equations, if the two roots are equal, then we have yn=(An+B)λny_n = (An+B)\lambda^nyn​=(An+B)λn for constants AAA, BBB. All that's left is to substitute in initial values of yny_nyn​ to find the constants, and we're done!

To deal with the pesky inhomogeneous term on the right-hand side, we once again resort to our tried-and-true, sophisticated, and wondrously erudite method of guessing.

If the right-hand side is an exponential function knk^nkn (k≠λk \neq \lambdak=λ), we guess knk^nkn or some multiple of this. (Be aware that kkk is a constant!)
