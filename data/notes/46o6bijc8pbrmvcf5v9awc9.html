Higher-Dimensional Determinants

Definition. Using the permutation sign and the precedent of the 3×33\times 33×3 determinant, define the determinant of a n×nn \times nn×n matrix AAA as

det⁡A=ϵj1j2...jnAj11Aj22...Ajnn\det A = \epsilon_{j_1 j_2 ... j_n}A_{j_1 1} A_{j_2 2} ... A_{j_n n}detA=ϵj1​j2​...jn​​Aj1​1​Aj2​2​...Ajn​n​

corresponding to the 3×33\times 33×3 and 2×22\times 22×2 determinants. The epsilon sign is only nonzero when j1,j2,...,jnj_1, j_2, ..., j_nj1​,j2​,...,jn​ are all chosen to be distinct, i.e. they are a permutation of {1,2,...,n}\{1,2,...,n\}{1,2,...,n}. As such, we can also rewrite the above as

det⁡A=∑σ(n)ϵ(σ(n))Aσ(1)1Aσ(2)2...Aσ(n)n\det A = \sum_{\sigma(n)}\epsilon(\sigma(n))A_{\sigma(1) 1}A_{\sigma(2) 2} ... A_{\sigma(n) n}detA=σ(n)∑​ϵ(σ(n))Aσ(1)1​Aσ(2)2​...Aσ(n)n​

summing over all possible permutations σ(n)\sigma(n)σ(n) of {1,2,3,...,n}\{1,2,3,...,n\}{1,2,3,...,n}.

Property. For any square matrix AAA, det⁡A=det⁡AT\det A = \det A^TdetA=detAT.

Proof. We have

det⁡AT=∑σ(n)ϵ(σ(n))A1σ(1)A2σ(2)...Anσ(n)\det A^T = \sum_{\sigma(n)}\epsilon(\sigma(n))A_{1 \sigma(1)}A_{2 \sigma(2)} ... A_{n \sigma(n)}detAT=σ(n)∑​ϵ(σ(n))A1σ(1)​A2σ(2)​...Anσ(n)​

with subscripts reversed from det⁡A\det AdetA. For every permutation σ(n)\sigma(n)σ(n), the product A1σ(1)A2σ(2)...Anσ(n)A_{1\sigma(1)}A_{2\sigma(2)}...A_{n\sigma(n)}A1σ(1)​A2σ(2)​...Anσ(n)​ is equivalent to Aρ(1)1Aρ(2)2...Aρ(n)nA_{\rho(1) 1}A_{\rho(2) 2}...A_{\rho(n) n}Aρ(1)1​Aρ(2)2​...Aρ(n)n​, as it is simply a rearrangement of the order of the terms. Thus, det⁡AT=det⁡A\det A^T = \det AdetAT=detA. (For instance, we could choose the permutation σ(σ−1(n))\sigma(\sigma^{-1}(n))σ(σ−1(n)).)

Proof. As det⁡A=det⁡A′\det A = \det A'detA=detA′, it suffices to prove this property for rows. Suppose that row jjj for j∈{1,2,...,n}j \in \{1,2,...,n\}j∈{1,2,...,n} is multiplied by λ\lambdaλ. Then

Property. Following from the above, det⁡(λA)=λndet⁡A\det(\lambda A)=\lambda^n \det Adet(λA)=λndetA for a n×nn\times nn×n matrix AAA.

Property. If A′A'A′ is AAA with two rows or columns interchanged, then det⁡A′=−det⁡A\det A' = - \det AdetA′=−detA.

Proof. Suppose that rows iii and jjj are swapped, with i<ji < ji<j; once again, due to det⁡A=det⁡AT\det A = \det A^TdetA=detAT, it suffices to prove this property for rows. Then we have

det⁡A′=∑σ(n)ϵ(σ(n))A1σ(1)′A2σ(2)′...(Aiσ(i)′)...(Ajσ(j)′)...Anσ(n)′\det A' = \sum_{\sigma(n)}\epsilon(\sigma(n))A'_{1 \sigma(1)}A'_{2 \sigma(2)} ... (A'_{i\sigma(i)})...(A'_{j\sigma(j)})...A'_{n\sigma(n)}detA′=σ(n)∑​ϵ(σ(n))A1σ(1)′​A2σ(2)′​...(Aiσ(i)′​)...(Ajσ(j)′​)...Anσ(n)′​

All the other A′A'A′s are left unchanged except Aiσ(i)′A'_{i\sigma(i)}Aiσ(i)′​, which becomes Ajσ(i)A_{j\sigma(i)}Ajσ(i)​ (and vice versa), so we have

det⁡A′=∑σ(n)ϵ(σ(n))A1σ(1)A2σ(2)...Aiσ(j)...Ajσ(i)...Anσ(n)\det A' = \sum_{\sigma(n)}\epsilon(\sigma(n))A_{1\sigma(1)}A_{2\sigma(2)}...A_{i\sigma(j)}...A_{j\sigma(i)}...A_{n\sigma (n)}detA′=σ(n)∑​ϵ(σ(n))A1σ(1)​A2σ(2)​...Aiσ(j)​...Ajσ(i)​...Anσ(n)​

Let the permutation {σ(1),σ(2),..,σ(j),...σ(i),...σ(n)}\{\sigma(1), \sigma(2),..,\sigma(j),...\sigma(i),...\sigma(n)\}{σ(1),σ(2),..,σ(j),...σ(i),...σ(n)} be denoted σ′\sigma'σ′. σ′\sigma'σ′ is equal to σ\sigmaσ with one transposition (σ(i)→σ(j)\sigma(i) \to \sigma(j)σ(i)→σ(j)), and so ϵ(σ′)=(−)ϵ(σ)\epsilon(\sigma')=(-)\epsilon(\sigma)ϵ(σ′)=(−)ϵ(σ). Thus det⁡A′=∑ϵ(σ′)...=−det⁡A\det A' = \sum \epsilon(\sigma') ... = - \det AdetA′=∑ϵ(σ′)...=−detA.

Property. Following from the above, if AAA has two identical rows or columns, det⁡A=0\det A = 0detA=0 because swapping the identical rows results in det⁡A′=det⁡A=−det⁡A\det A' = \det A = - \det AdetA′=detA=−detA.
