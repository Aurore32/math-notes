Tensors are Multilinear Maps

What are multilinear maps?

Sometimes you flip to a new chapter in your math textbook, stare at it, get angry, and then when the overwhelming tidal wave of rage and devastation finally recedes from your body all that's left is for you to wonder "why?" Why did people look at this:

Definition. A multilinear map fff is an extension of linear maps to a domain of multiple vectors at once: it maps nnn vectors v1,...,vn\mathbf{v_1, ..., v_n}v1​,...,vn​, each belonging to (potentially distinct) vector spaces V1,V2,...,VnV_1, V_2, ..., V_nV1​,V2​,...,Vn​, to a vector w\mathbf{w}w in vector space WWW. Denote this as f(v1,...,vn)=w,f:V1×V2×...×Vn→Wf(\mathbf{v_1, ..., v_n}) = \mathbf{w}, f: V_1 \times V_2 \times ... \times V_n \to Wf(v1​,...,vn​)=w,f:V1​×V2​×...×Vn​→W.

The "multi" part of "multilinear" comes from the fact that fff has linearity in each of the nnn variables it accepts as its input. A regular linear map A(x)A(\mathbf{x})A(x) is linear because it satisfies the following condition:

A(λx+μy)=λA(x)+μA(y)A(\lambda \mathbf{x} + \mu \mathbf{y}) = \lambda A(\mathbf{x}) + \mu A(\mathbf{y})A(λx+μy)=λA(x)+μA(y)

Analogously, for a multilinear map this linearity condition holds for each of v1,...,vn\mathbf{v_1}, ..., \mathbf{v_n}v1​,...,vn​:

f(v1,...,λvi+μui,...,vn)=λf(v1,...,vi,...vn)+μf(v1,...,ui,...,vn)\begin{aligned} f(\mathbf{v_1, ..., \lambda v_i + \mu u_i, ..., v_n}) \\ = \lambda f(\mathbf{v_1, ..., v_i, ... v_n}) + \mu f(\mathbf{v_1, ..., u_i, ..., v_n}) \end{aligned}f(v1​,...,λvi​+μui​,...,vn​)=λf(v1​,...,vi​,...vn​)+μf(v1​,...,ui​,...,vn​)​

for every one of i=1,...,ni = 1, ..., ni=1,...,n. For instance, a bi-linear map f:U×V→Wf: U \times V \to Wf:U×V→W for vector spaces U,V,WU, V, WU,V,W has the property that, for four vectors a,b,c,d\mathbf{a, b, c, d}a,b,c,d,

This gives us the tools to construct a coordinate-independent definition of tensors, based on multilinear maps. We focus on "square" (or cube, or hypercube, etc.; I don't exactly know the fifth-dimensional lingo the Generation Ω\OmegaΩ hyperkids are using today) tensors for now.

In order to show that this is indeed equivalent to the previous definition of a tensor we have given, we proceed towards showing that 1) multilinear maps are independent of basis, and 2) multilinear maps transform like tensors. This necessitates the introduction of some new (and improved) tensor operators, so buckle up!

Tensor operations

Proposition. Preservation of the tensor transformation law.

There is one commonality that unites all the following tensor operations: they all suck given two tensors SSS and TTT, they will all output another tensor, be it S+TS + TS+T (addition), S⊗TS \otimes TS⊗T (lamps in an electric circuit tensor multiplication), or anything of the like - meaning that it preserves the tensor transformation law.

Linear operations on tensors

Tensor multiplication

Definition. Given two tensors SSS and TTT of ranks ppp and qqq respectively, define the tensor product S⊗TS \otimes TS⊗T outputting a tensor of rank p+qp + qp+q on a component-by-component basis as

(S⊗T)i1i2...ipj1...jq=Si1i2...ipTj1...jq(S\otimes T)_{i_1 i_2 ... i_p j_1 ... j_q} = S_{i_1 i_2 ... i_p} T_{j_1 ... j_q}(S⊗T)i1​i2​...ip​j1​...jq​​=Si1​i2​...ip​​Tj1​...jq​​

We make the important clarification that this is defined for tensors which aren't of the same "dimensions" (e.g. a 2×22\times 22×2 rank-222 tensor and a 1×31\times 31×3 rank-111 tensor).

This is not matrix multiplication, or the dot product, or anything we've seen before as we know it; it's something entirely new, defined between any two tensors. When we're just dealing with vectors and matrices, it's actually not all that hard to visualize. Consider the tensor product between

Symmetric and anti-symmetric matrices

Definition. A nnn-tensor Tijp...qT_{ijp ...q}Tijp...q​ is symmetric about a pair of indices iii and jjj, again arbitrarily chosen, if Tijp...q=Tjip...qT_{ijp...q} = T_{jip...q}Tijp...q​=Tjip...q​ (swapping the indices); call TTT anti-symmetric about the same pair of indices if instead Tijp...q=−Tjip...qT_{ijp...q} = -T_{jip...q}Tijp...q​=−Tjip...q​.

If a tensor is symmetric (or anti-symmetric) in all possible pairs of indices, call it totally symmetric (or anti-symmetric).

Proposition. In R3\mathbb{R^3}R3, there are no nonzero rank-444 or higher tensors that are totally anti-symmetric.

Proof.

Suppose for the sake of contradiction that Ti1i2...ipT_{i_1 i_2 ... i_p}Ti1​i2​...ip​​ is an anti-symmetric rank-ppp tensor in R3\mathbb{R^3}R3, with p>3p > 3p>3 and ij=1,2,3i_j = 1, 2, 3ij​=1,2,3 for all j=1,2,...,pj = 1, 2, ..., pj=1,2,...,p. By the Pigeonhole Principle, we have 333 numbers (1,2,31, 2, 31,2,3) and ppp boxes to fit them into with p>3p > 3p>3; sooner or later there will be a repeated index.

Without loss of generality, let ij=iki_j = i_kij​=ik​. Thus

Tijik...ip=Tikij...ipT_{i_j i_k ... i_p} = T_{i_k i_j ... i_p}Tij​ik​...ip​​=Tik​ij​...ip​​

by ij=iki_j = i_kij​=ik​, and

Tijik...ip=−Tikij...ipT_{i_j i_k ... i_p} = -T_{i_k i_j ... i_p}Tij​ik​...ip​​=−Tik​ij​...ip​​

Tijik...ip=0.T_{i_j i_k... i_p} = 0.Tij​ik​...ip​​=0.

Note that this proof can be extended naturally to rank-ppp or higher tensors in Rn\mathbb{R^n}Rn with p>np > np>n.
