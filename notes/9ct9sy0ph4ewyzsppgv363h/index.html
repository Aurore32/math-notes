Point sources and the Delta function

As mentioned before, an important caveat to the isotropic solution for the Laplace equation in R3\mathbb{R^3}R3 - which takes the form ψ(r)=α−βr−1\psi(r) = \alpha - \beta r^{-1}ψ(r)=α−βr−1 - is the fact that it diverges at the origin, where r=0r = 0r=0; at that point, it can no longer be construed as a valid solution to the Laplace equation. In that case, then, 1) what does r=0r=0r=0 mean in a physical sense, and 2) what equation does the fundamental solution solve at that point?

The answer to the first question is slightly easier. A gravitational field around an infinitely small point mass, for instance, would satisfy the Laplace equation everywhere except for the point where the mass is located; at every other point, the mass density of the point mass is zero. As such, r=0r = 0r=0 in the context of a solution to the Laplace equation can be interpreted as gravitational potential near a point mass - all the density concentrated in a single point, zero density everywhere else, and infinite potential/energy required to bring another mass to that point as a result.

In particular, the Delta function's presence represents a point source for the potential: one that exists only at a single point and nowhere else.

An integral solution for the Poisson equation

We understand from the above that the isotropic solution

ψ(r)=λ/4πr\psi(r) = \lambda/4\pi rψ(r)=λ/4πr

to the Laplace equation models a point source of strength λ\lambdaλ placed at the origin; if instead the point source is placed at a point x′\mathbf{x'}x′, then by extension we have

ψ(x)=λ4π∣x−x′∣.\psi(\mathbf{x}) = \frac{\lambda}{4\pi |\mathbf{x - x'}|}.ψ(x)=4π∣x−x′∣λ​.

Now suppose that instead of a single point source, we have many, many different point sources scattered about a closed region VVV; each has a different strength λ\lambdaλ - enough to justify our use of a function ρ\rhoρ to represent the strength of the point source placed at point x′\mathbf{x'}x′:

λ=ρ(x′).\lambda = \rho(\mathbf{x'}).λ=ρ(x′).

Eventually, when we have an infinite number of point sources that cover VVV in its entirety and ρ\rhoρ becomes a smooth, continuous function, the point sources collect together to become an object bounded by VVV with density ρ(x′)\rho(\mathbf{x'})ρ(x′) at each point:

ψ(x)=∫Vρ(x′)4π∣x−x′∣ dV.\psi(\mathbf{x}) = \int_V \frac{\rho(\mathbf{x'})}{4\pi |\mathbf{x - x'}|}\ dV.ψ(x)=∫V​4π∣x−x′∣ρ(x′)​ dV.

where, importantly, the variable being integrated through in the volume integral is x′\mathbf{x'}x′, not x\mathbf{x}x. This ultimately leads to one final claim:

Theorem. The amalgamated object ψ(x)\psi(\mathbf{x})ψ(x), represented by the above volume integral of point sources, is a solution to the Poisson equation ∇2ψ=−ρ.\nabla^2 \psi = -\rho.∇2ψ=−ρ.

The surface-level dark-magic trickery of what we're doing here belies a simple explanation for how this might have occurred: the Poisson equation ∇2ψ=−ρ(x)\nabla^2 \psi = -\rho(\mathbf{x})∇2ψ=−ρ(x) describes a potential function ψ\psiψ which arises from a source of density ρ\rhoρ at points x∈R3\mathbf{x} \in \mathbb{R^3}x∈R3. Therefore, if at each infinitesimal point some function ψ\psiψ has a point source placed there with strength ρ\rhoρ, and if it smells like the potential function of ρ\rhoρ and talks like the potential function of ρ\rhoρ, then it is the potential function of ρ\rhoρ.
