Definition. Define matrix multiplication as follows:

  1. Matrix multiplication is only valid between a n×mn \times mn×m matrix and a m×pm \times pm×p matrix for any positive integers nnn, mmm, ppp: the number of columns of the first matrix must match the number of rows of the second.
  2. Let AAA be a n×mn\times mn×m matrix and BBB be m×pm \times pm×p; write CCC as the matrix resulting from the multiplication ABABAB. Then CCC has dimensions n×pn \times pn×p, and Cij=∑k=1mAikBkjC_{ij} = \sum_{k=1}^{m} A_{ik}B_{kj}Cij​=∑k=1m​Aik​Bkj​. Alternatively, it is sum of the product of the elements of the iiith row of AAA and the jjjth column of BBB (which have the same length, as we insisted).

Note that matrix multiplication often does not satisfy commutativity: AB≠BAAB \neq BAAB=BA.

Linear maps

Definition. Let AAA, BBB be sets. A map TTT (not necessarily linear) from AAA to BBB assigns each element x∈Ax \in Ax∈A a unique x′∈Bx' \in Bx′∈B. We write

T:A→B and/or x→x′=T(x)T: A \to B \text{ and/or } x\to x' = T(x)T:A→B and/or x→x′=T(x)

Definition. AAA is the domain of TTT, BBB the range (codomain), T(x)=x′T(x)=x'T(x)=x′ the image of xxx under TTT, and A→T(A)A \to T(A)A→T(A) is the image of AAA under TTT (i.e. the entire set AAA transformed to T(A)T(A)T(A)).

While it is true that T(A)T(A)T(A) is a subset of BBB, BBB may have elements that are not in T(A)T(A)T(A).

Definition. (Linear maps). Let VVV and WWW be real vector spaces (Rn\mathbb{R^n}Rn and Rm\mathbb{R^m}Rm respectively). TTT is a linear map from VVV to WWW if it satisfies the following properties:

  1. T(a+b)=T(a)+T(b)T(\mathbf{a+b}) = T(\mathbf{a}) + T(\mathbf{b})T(a+b)=T(a)+T(b), for vectors a,b∈Rn\mathbf{a, b}\in \mathbb{R^n}a,b∈Rn. (Linearity)
  2. T(λa)=λT(a)T(\lambda \mathbf{a})=\lambda T(\mathbf{a})T(λa)=λT(a) for a scalar λ∈R\lambda \in \mathbb{R}λ∈R. (Scalar multiplicity)

Combined together, we have T(λa+μb)=λT(a)+μT(b)T(\lambda \mathbf{a} + \mu \mathbf{b})=\lambda T(\mathbf{a})+ \mu T(\mathbf{b})T(λa+μb)=λT(a)+μT(b): TTT can be distributed across a linear combination.

Proposition. T(V)T(V)T(V) is a subspace of WWW.

Proof. By the above properties, T(V)T(V)T(V) is a subset of WWW and is closed under any linear combinations. Thus it is a subspace by definition.
